CBT 1.7.2 (Test V7.2) Operator Help

usage manual

Fast start

  • Choose role and environment profile, then press Start Core.
  • Run Self Tests and Field Matrix before field deployment.
  • Use BLE Protocol Scan for known VitalGuard nodes; use Passive BLE Scan only after the consent modal and only for authorized humanitarian deployments.
  • Use the SOS board, audio beacon, flash beacon, vibration beacon, and QR/SVG SOS card as fallback communication channels.
  • Export sanitized audit evidence after drills; run Emergency 7-Pass Wipe before the device leaves operator custody.

Safety claims

VitalGuard CBT 1.7.2 (Test V7.2) improves local detection and coordination probability. It does not guarantee detection, rescue, exact location, medical outcome, browser background operation, or BLE advertisement transmit from a normal browser.

Privacy rules

No external servers, no persistent identity profiles, no biometric recognition, no raw audio persistence, short retention windows, recursive sanitization, and one-gesture best-effort deletion.
